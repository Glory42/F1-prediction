Fueling F1 Predictions: How It Works (CEN302)
This course project, "Formula 1 Result Forecast," predicts race outcomes using qualifying data. Advanced algorithms and AI models analyze past races, driver performance, and track details. Powerful Python libraries and our backend (FastAPI) evaluate even tiny time differences for high-accuracy predictions. We use reliable APIs (fastf1) for data, stored in PostgreSQL. The aim of this project is to offer fans data-driven forecasts pre-race, enhancing the excitement.
